# **App Name**: SnazzPay

## Core Features:

- Shopify Integration: Integrate with Shopify store www.snazzify.co.in to fetch order details.
- eMandate Setup: Initiate Razorpay eMandate for recurring payments.
- Secure COD Processing: Securely process Cash on Delivery (COD) orders using Razorpay.
- Mandate Management: User-friendly interface for managing mandates and payment schedules.
- Payment Notifications: Real-time payment status updates and notifications.

## Style Guidelines:

- Primary color: Deep Indigo (#663399) to convey trust and security.
- Background color: Light Lavender (#E6E0EB), a desaturated version of the primary color, to create a calm and professional backdrop.
- Accent color: Soft Purple (#804BB3) as an analogous hue to subtly highlight important elements.
- Font: 'Inter', a grotesque-style sans-serif font for a modern and neutral appearance, suitable for headlines or body text
- Use clean and simple icons for payment methods and status indicators.
- Clean and intuitive layout with clear calls to action for managing payments and mandates.
- Subtle animations for payment confirmations and loading states.
